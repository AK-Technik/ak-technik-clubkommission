# AK Technik Clubkommission

Shared knowledge base and problem-solving platform for technicians at Berlin clubs and venues.

This repository is part of the **Techniker-Stammtisch** organised by the Clubkommission. The goal: make collective knowledge accessible, avoid duplicate work, and learn from each other.

## 💬 How to contribute

### Report a problem / ask a question
→ [**Create a new Issue**](../../issues/new/choose)

Three templates are available:
- **🐛 Problem / Fehlverhalten** – something isn't working as expected
- **✅ Lösung dokumentieren** – share a working setup or solution
- **💡 Frage / Diskussion** – open questions for the group

### Discussions
For open conversations without a specific problem:
→ [**Discussions**](../../discussions)

### Add documentation
1. Navigate to the relevant folder under `/docs`
2. Click **Add file** → **Create new file** (Markdown `.md`) or **Upload files** (PDF, images)
3. Click **Commit changes** — done
